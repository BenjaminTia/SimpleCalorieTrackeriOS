# Simple Calorie Tracker

A lightweight iOS utility app for tracking daily calorie intake, built with Swift and Xcode.

## Overview

Simple Calorie Tracker is a personal iOS app designed to make logging daily food intake quick and straightforward. The app allows users to add meals with calorie values, view their daily total, and stay on top of their nutrition goals — all within a clean, minimal interface.

## Features

- Add and log meals with individual calorie counts
- View running daily calorie total
- Simple and intuitive UI designed for everyday use
- Lightweight — no account or internet connection required

## Tech Stack

| Technology | Usage |

## Getting Started

### Requirements
- macOS with Xcode installed (Xcode 14+ recommended)
- iOS Simulator or a physical iPhone/iPad

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/BenjaminTia/Simple-Calorie-Tracker.git
   ```
2. Open `Simple Calorie Tracker.xcodeproj` in Xcode
3. Select a simulator or connected device
4. Press `Cmd + R` to build and run
